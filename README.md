# Sonar Rock vs Mine Prediction

## Overview

This project uses Machine Learning to classify sonar signals as either Rock (R) or Mine (M). The model is trained using Logistic Regression on the Sonar Dataset.

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* Google Colab

## Algorithm

* Logistic Regression

## Model Performance

* Training Accuracy: 83.42%
* Testing Accuracy: 76.19%

## Dataset

The Sonar Dataset contains 60 numerical features and a target label:

* R = Rock
* M = Mine

## Project Workflow

1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Accuracy Evaluation
6. Prediction
